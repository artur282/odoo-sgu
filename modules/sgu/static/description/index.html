SGU - Sistema de Gestión Universitaria

El módulo SGU (Sistema de Gestión Universitaria) es una solución integral para la gestión de instituciones educativas de nivel superior.

Características Principales

Configuración Institucional

Gestione la estructura de su institución con sedes, carreras, niveles académicos y áreas.

  • Múltiples sedes
  • Diferentes modalidades de estudio
  • Definición de niveles académicos
  • Organización por áreas académicas

Gestión de Programas Académicos

Administre los programas académicos y su estructura curricular.

  • Administración de carreras
  • Gestión de pensums
  • Control de materias y unidades de crédito
  • Asignación de carreras a sedes

Procesos de Admisión

Automatice el proceso completo de admisión estudiantil.

  • Importación de datos OPSU
  • Registro de aspirantes
  • Preinscripción e inscripción
  • Asignación de secciones

Control Académico

Mantenga el control de los periodos y procesos académicos.

  • Gestión de periodos académicos
  • Programación de procesos
  • Asignación de secciones
  • Generación de horarios

Documentación

Genere automáticamente documentación oficial.

  • Constancias de inscripción
  • Constancias de estudio

Gestión de Usuarios

Control de acceso y permisos basados en roles.

  • Super Usuario
  • Usuario Admin
  • Operador Admin
  • Estudiante

Funcionamiento

El sistema permite estructurar de manera jerárquica la organización educativa, desde la definición de la institución, sus sedes, áreas, carreras y pensums, hasta el registro y control de estudiantes, desde su admisión a la generación de documentos oficiales.

El flujo principal contempla:

  1. Configuración de la estructura institucional
  2. Recepción de aspirantes OPSU o casos aprobados
  3. Preinscripción de aspirantes
  4. Verificación de documentos
  5. Inscripción formal
  6. Conversión de aspirantes a estudiantes regulares
  7. Gestión académica continua

Requerimientos

El módulo SGU requiere:

  • Odoo 18.0 o superior
  • Módulo base de Odoo

Para soporte o más información, contacte a su proveedor de servicios Odoo.
